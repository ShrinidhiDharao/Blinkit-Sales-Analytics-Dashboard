# Blinkit-Sales-Analytics-Dashboard
🛒 Blinkit Sales Analytics Dashboard

An interactive Power BI dashboard built to analyze Blinkit's sales performance, customer ratings, outlet characteristics, and product distribution. The dashboard enables users to explore business insights through dynamic filters, KPI tracking, and interactive visualizations.

📖 Short Description / Purpose

The Blinkit Sales Analytics Dashboard provides a comprehensive overview of retail sales performance by combining key business metrics with interactive visualizations. It allows users to analyze sales across different outlet sizes, locations, outlet types, and product categories to support data-driven decision making.

Designed with an intuitive layout and navigation, the dashboard simplifies complex sales data into meaningful business insights.

🛠️ Tech Stack

The dashboard was built using the following tools and technologies:

📊 Power BI Desktop – Dashboard development and data visualization.
📂 Power Query – Data cleaning and transformation.
🧠 DAX (Data Analysis Expressions) – Calculated measures and KPIs.
🔗 Data Modeling – Relationship management for interactive reporting.
🎨 Interactive Visuals – KPI Cards, Bar Charts, Donut Charts, Line Charts, Tables, Slicers, and Navigation Buttons.
📁 File Format – .pbix for development and .png for dashboard preview.
📂 Data Source

Source: Blinkit Grocery Sales Dataset

The dataset contains information on grocery sales, product categories, outlet locations, outlet sizes, customer ratings, outlet establishment years, and product attributes. It is used to analyze business performance across multiple operational dimensions.

✨ Features / Highlights
• Business Problem

Retail businesses generate large volumes of sales data, making it difficult to identify performance trends, compare outlet efficiency, and understand customer purchasing behavior through raw data alone.

• Goal of the Dashboard

To develop an interactive dashboard that enables users to monitor key business metrics, compare outlet performance, analyze product sales, and uncover actionable insights through dynamic filtering and visualization.

• Walkthrough of Key Visuals
KPI Cards – Monitor Total Sales, Average Sales, Total Items Sold, and Average Rating.
Interactive Filter Panel – Filter data by Outlet Location, Outlet Size, and Item Type.
Fat Content Analysis – Compare sales contribution of Low Fat and Regular products.
Item Type Analysis – Identify top-performing product categories.
Outlet Establishment Trend – Analyze sales performance over time.
Outlet Size & Location Analysis – Compare sales across outlet sizes and city tiers.
Outlet Type Table – View detailed performance metrics for each outlet category.
Navigation Buttons – Seamless navigation between dashboard views.
• Business Impact & Insights
Identifies high-performing outlets and product categories.
Supports inventory and sales strategy decisions.
Enables comparison across outlet sizes and locations.
Provides an interactive, user-friendly reporting experience for business analysis.
